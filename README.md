Eclipse-on-linux
================

A utility script to install a Eclipse JDT on linux.
===

Installation:
```sh
$ git clone https://github.com/SBSTP/Eclipse-on-linux.git
$ cd Eclipse-on-linux
$ sudo ./install.sh
```  

Uninstallation:
```sh
$ cd <installation directory> (e.g. /usr/local/eclipse)
$ sudo ./uninstall.sh
```
